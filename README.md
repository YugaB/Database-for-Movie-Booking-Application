# CMPE226

## Movie Booking Application

### To Run Application 

```
cd cmpe226
pip install -r requirements.txt
python server.py
```

---

### Technologies:

#### Front-end:
```
- Jinja
- HTML
```

#### Back-end:
```
- Python
- Flask
```

#### Database:
```
- MySQL (AWS RDS)
- MongoDB (mLab)
```
